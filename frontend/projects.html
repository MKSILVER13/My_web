LLM-Compiler Based Agentic RAG System for Legal QA

13th Inter IIT Tech Meet, IIT Bombay (Nov 2024 - Dec 2024)

  • Leveraged LLM-Compiler for parallel task execution, dynamic replanning, task-specific retrieval tools, and scalable task management with DAGs, reducing costs by decoupling reasoning from execution.
  • Used a recursive character text splitter for long-context legal documents, integrated VoyageAI’s Voyage-Law2 embedding model, and developed a custom auto-merging retriever with Cohere’s reranker to improve accuracy.
  • Integrated Anthropic’s Claude models as a fallback for OpenAI API failures and developed a self fine-tuned beam retriever to improve robustness in retrieval workflows.
  • Incorporated metadata like page numbers, paragraphs, document name, and in-context citations into prompts for reliable legal QA and citation.
  • Achieved 86% accuracy on the CUAD dataset (contract law) and 84% accuracy on the AILA dataset (case law).

Hub Quest: Delivery Route Optimization

HackVerse 2025 (Apr 2025 - Apr 2025)

  • Developed a delivery route optimization system to find efficient paths for package deliveries, considering hub-based routing and fuel constraints.
  • Implemented a modified Floyd-Warshall algorithm for pre-calculating all-pairs shortest paths, storing complex state information including fuel usage, visited hubs, and delivered packages.
  • Designed and implemented a heuristic search algorithm (findOptimalPath) using a stack-based approach with backtracking to construct and optimize delivery routes, maximizing deliveries within fuel limits.
  • Managed vehicle fuel capacity constraints and refueling logic by integrating it into path state merging and validation processes.
